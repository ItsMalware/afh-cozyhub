# AFH Branding Mode - Founder Decision Memo
Date: 2026-03-10 (ET)
Owner: Founder + Prime

## Decision Goal
Lock MVP scope for Branding Day Mode so execution can start immediately with low risk and high usefulness.

## Scope Decision (MVP)
Approved MVP features:
1. Weekly Action Queue
2. AI Insight Card (what changed / why it matters / what to do next)
3. Funnel Snapshot
4. Channel Mix
5. Moments Snapshot

Deferred to post-MVP:
- Deep moments matrix drilldowns
- Advanced association graphing
- Multi-model forecast comparison

## Why this scope
- Preserves low-clutter UX for daily execution.
- Delivers immediate actionability for branding days.
- Uses existing Notion + NotebookLM foundations with manageable implementation effort.

## Non-Negotiables
- Dashboard must remain focus-first and non-overwhelming.
- No fake agent statuses; connector truthfulness required.
- Prediction outputs must include confidence + assumptions.
- All work logs require Hours_cmmd_hub.

## 6-Week Delivery Commit
- Week 1: data model and storage contract
- Week 2: backend synthesis and orchestration
- Week 3: UI shell + first widgets
- Week 4: complete all 5 MVP widgets
- Week 5: prediction panel + company drilldowns
- Week 6: risk hardening, QA, release gate

## Success Criteria
- Founder can open Branding Day Mode and choose the next best action in under 60 seconds.
- Monday-night refresh produces Tuesday-ready summary.
- No dashboard overload regressions.

## Final Decision
MVP scope is approved as listed above. Begin implementation with AFH-RESEARCH-019 immediately.
